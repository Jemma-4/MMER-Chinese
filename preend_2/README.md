## 前端配置与用户说明

#### 相关配置

+ 依赖安装

  ```
  yarn install
  ```

+ 项目运行

  ```
  yarn serve
  ```

#### 用户使用过程

+ 以文本|语音的方式输入对问题的回答
+ 对回答长句进行切分，模型对切分后的短文本列表进行细粒度情绪分析并展示初步分类结果
+ 可以对分类结果进行修正，上传正确情绪标签
